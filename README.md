# Phishing Website Detector

A Machine Learning-based cybersecurity project that identifies phishing websites by analyzing URL characteristics and patterns. The system is trained on phishing and legitimate website datasets and provides real-time predictions through a user-friendly Flask web interface.

## Features

* Detects phishing and legitimate websites
* URL-based feature extraction
* Machine Learning classification
* User-friendly web interface using Flask
* Fast and accurate predictions
* Helps improve online security awareness

## Technologies Used

* Python
* Flask
* Pandas
* NumPy
* Scikit-learn
* HTML/CSS

## Dataset

The model is trained using phishing URL datasets containing both malicious and legitimate website URLs.

## Use Cases

* Cybersecurity awareness
* Educational projects
* Website safety verification
* Research on phishing detection
